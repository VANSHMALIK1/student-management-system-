# Student Management System (Java + MySQL)

## 📌 Project Overview
This is a console-based Student Management System developed using Java and MySQL with JDBC connectivity. The project demonstrates basic CRUD (Create, Read, Delete) operations and follows a structured DAO (Data Access Object) architecture.

## 🚀 Features
- Add new student records
- View all students
- Delete student by ID
- Structured database connectivity using JDBC
- Organized OOP-based class structure

## 🛠 Technologies Used
- Java
- MySQL
- JDBC
- Object-Oriented Programming (OOP)

## 🗂 Project Structure
- DBConnection.java – Handles database connectivity
- Student.java – Student model class
- StudentDAO.java – Contains database operations
- Main.java – Console-based user interface

## 🗄 Database Schema
Database Name: student_db  
Table Name: students  

Fields:
- id (Primary Key, Auto Increment)
- name
- email
- course

## ▶ How to Run
1. Install Java (JDK 8+)
2. Install MySQL Server
3. Create database and table as per schema
4. Update database username and password in DBConnection.java
5. Compile and run Main.java

## 📖 Learning Outcome
This project helped in understanding database integration, JDBC connectivity, and implementation of CRUD operations in a structured Java application.
